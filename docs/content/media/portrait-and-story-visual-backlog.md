# Portrait And Story Visual Backlog

## Purpose

This backlog turns the user's visual direction into durable repository work.

The site should become more engaging through real faces, documentary artifacts,
institutional imagery, and story-led captions, but it should do so without
using fake portraits or fabricated historical scenes.

## Core Rule

For real people, institutions, and turning points, prefer sourced visuals with
clear provenance.

Generated imagery may support conceptual teaching, but it should not replace
historical portraits, documentary photos, scanned proposals, or real lab and
artifact images when those are available.

## Portrait Rules

- use sourced portraits or institutional headshots for real people
- never use generated portraits for historical figures or living researchers
- record provenance, rights uncertainty, and editorial rationale in media
  records before live use
- choose portraits that reinforce the story being told rather than generic press
  glamour shots when better documentary options exist

## Story-Led Caption Rules

- captions should explain why the person, institution, or turning point matters
- emphasize the human story, constraint, or decision visible in the image
- avoid filler captions that only repeat the subject's name
